# Little Psychologist Bot

Телеграм-бот для психологического отбора кандидатов в сферу сервиса. Проводит структурированное поведенческое интервью по 4-уровневой модели профессионально важных качеств (ПВК) и формирует аналитический отчёт.

## Стек

- **Runtime:** Node.js + TypeScript
- **Framework:** NestJS
- **Bot:** grammY (Telegram Bot API)
- **LLM:** Claude (Anthropic) / OpenAI / GigaChat — переключаемые провайдеры
- **Observability:** Langfuse
- **Test:** Vitest
- **Linter/Formatter:** Prettier + Husky + lint-staged

## Структура проекта

```
src/
├── app/            # Корневой модуль NestJS
├── assessment/     # Логика оценки кандидата (сервис, промпты, конфиг)
├── chat/           # Обработка сообщений Telegram
├── i18n/           # Все текстовые строки (ru / en)
└── llm/            # Абстракция над LLM-провайдерами
```

## Команды

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки (.env.dev)
npm start

# Запуск в продакшне (.env.prod)
npm run start:prod

# Сборка TypeScript
npm run build

# Тесты
npm test
```

## Переменные окружения

Создай `.env.dev` (для разработки) и `.env.prod` (для продакшна):

```env
BOT_TOKEN=           # Telegram Bot Token (от @BotFather)
CLAUDE_API_KEY=      # Anthropic API Key
OPENAI_API_KEY=      # OpenAI API Key
DEEPSEEK_API_KEY=    # DeepSeek API Key
GIGACHAT_API_KEY=    # GigaChat API Key
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/assess` | Начать оценку кандидата |
| `/cancel` | Отменить текущую оценку |

## Логика оценки

Бот задаёт 4 поведенческих вопроса, каждый из которых диагностирует один уровень модели:

| Уровень | ПВК |
|---------|-----|
| I | Физическая выносливость и точность |
| II | Стрессоустойчивость и саморегуляция |
| III | Социальная перцептивность и эмпатия |
| IV | Этическая позиция и ценностная направленность |

Перед каждым вопросом кандидату показывается инструкция по структуре ответа (ситуация → эмоция → действие → результат). Если ответ недостаточно развёрнут — бот задаёт один уточняющий вопрос. По завершении LLM формирует развёрнутый отчёт с оценками по каждому уровню и итоговой рекомендацией.
